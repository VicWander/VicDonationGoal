DONATION GOAL WIDGET — СТАРТОВЫЙ ПРОЕКТ

Что это:
Локальный виджет донат-гола для OBS.
Пока без DonationAlerts. На первом этапе донаты добавляются вручную через админку.
Потом DonationAlerts можно подключить отдельным модулем, не переписывая сам виджет.

Как запустить на Windows:

1) Распакуй папку donation_goal_widget в удобное место.
Например:
C:\Users\ТВОЁ_ИМЯ\Desktop\donation_goal_widget

2) Открой папку.

3) Дважды нажми start.bat.

4) Должно открыться чёрное окно с текстом:

Donation Goal Widget запущен
Виджет для OBS: http://127.0.0.1:3333/widget.html
Админка:        http://127.0.0.1:3333/admin.html

5) Открой в браузере:
http://127.0.0.1:3333/admin.html

6) Там можно:
- изменить название цели;
- изменить текущую сумму;
- изменить сумму цели;
- добавить тестовый донат;
- отменить последний донат;
- сбросить прогресс.

7) Открой в браузере:
http://127.0.0.1:3333/widget.html

Это сама полоска.

Как добавить в OBS:

1) Открой OBS.
2) В источниках нажми плюс.
3) Выбери "Браузер" / "Browser".
4) Создай новый источник.
5) В URL вставь:
http://127.0.0.1:3333/widget.html
6) Ширина: 800
7) Высота: 220
8) Нажми OK.

Важно:
Пока сервер запущен, виджет работает.
Если закрыть чёрное окно сервера, виджет перестанет обновляться.

Где менять внешний вид:
public/style.css

Главные размеры:
.goal-card {
  width: 760px;
}

Главные цвета полоски:
.goal-fill {
  background: linear-gradient(90deg, #55d8ff, #9c76ff);
}

Где хранятся данные:
state.json — текущая цель
donations.json — история донатов

Если всё сломалось:
1) Закрой сервер.
2) Удали state.json и donations.json.
3) Запусти start.bat заново.
Файлы создадутся снова.
