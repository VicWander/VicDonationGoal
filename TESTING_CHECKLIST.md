# Release testing checklist

Перед публикацией v1.0.0 желательно проверить проект в отдельной папке.

## Чистый запуск

- [ ] Распаковать релизный архив в новую папку.
- [ ] Запустить `00_INSTALL_ALL.bat`.
- [ ] Убедиться, что создались локальные конфиги из `.example.json`.
- [ ] Запустить `02_START_SERVER_ONLY.bat`.
- [ ] Открыть `http://127.0.0.1:3333/admin.html`.
- [ ] Открыть `http://127.0.0.1:3333/widget.html`.
- [ ] Открыть `http://127.0.0.1:3333/docs/index.html`.

## Админка

- [ ] Добавить ручной донат.
- [ ] Проверить, что сумма попала в сбор.
- [ ] Выключить `В сбор` у доната.
- [ ] Проверить пересчёт суммы.
- [ ] Проверить поиск и фильтры.
- [ ] Проверить экспорт CSV.
- [ ] Проверить светлую/тёмную тему.
- [ ] Проверить ручные курсы валют.

## Streamer.bot

- [ ] Включить HTTP Server в Streamer.bot.
- [ ] Указать action ID в `streamerbot_config.json`.
- [ ] Запустить `04_START_STREAMERBOT_ONLY.bat`.
- [ ] Запустить `05_TEST_STREAMERBOT.bat`.
- [ ] Проверить кнопку `Повт.` в истории донатов.

## DonationAlerts

- [ ] Создать OAuth-приложение.
- [ ] Заполнить `da_config.json`.
- [ ] Открыть `/auth/donationalerts/start`.
- [ ] Проверить `/api/da/status`.

## Donatty

- [ ] Заполнить `donatty_config.json`.
- [ ] Запустить `03_START_DONATTY_ONLY.bat`.
- [ ] Проверить тестовый донат.

## Финальная проверка безопасности

- [ ] В архиве нет `da_tokens.json`.
- [ ] В архиве нет настоящего `da_config.json`.
- [ ] В архиве нет настоящего `donatty_config.json`.
- [ ] В архиве нет настоящего `streamerbot_config.json`.
- [ ] В архиве нет `donations.json`.
- [ ] В архиве нет `state.json`.
